# Admissions Funnel Excel Dashboard

## Overview
This project analyzes an edtech admissions funnel using Excel to identify lead drop-offs, recovery opportunities, and conversion strength across applicant stages.

The dashboard consolidates CRM call dispositions into actionable funnel states:
- **Recoverable** (Busy / Call Later)
- **Lost** (DNP)
- **Converted** (Test Booked)

## Key Business Questions Answered
- At which applicant stage are the maximum leads lost?
- Which stages have high recovery potential?
- Where should counseling and follow-up efforts be focused?

## Dashboard Features
- 100% stacked column chart showing **Lost vs Recoverable vs Converted** by applicant stage
- Interactive slicers:
  - Lead Age (months)
  - Lead Category
  - City
- Pivot-based aggregation for fast filtering and analysis

## Key Insights
- The highest funnel leakage occurs **post-registration**, driven primarily by DNPs
- Leads attending L3 sessions show strong recovery potential
- Conversion is extremely strong once users reach the test scheduling stage

## Tools Used
- Google Sheets
- Pivot Tables
- Pivot Charts
- Slicers
- Funnel-based metric grouping
